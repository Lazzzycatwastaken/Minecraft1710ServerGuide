Minecraft Forge 1.7.10 Server Setup

Quick guide to get your modded server running

1. Download Forge

Search "Forge 1.7.10" and get the installer from the official site.

The Forge download page - click the installer button
Hit "Skip" on any ad pages that pop up

2. Install Server

Run the installer, select Install Server, pick an empty folder.

3. First Run

Go to your server folder and double-click minecraft_server.1.7.10.jar

Click on minecraft_server.1.7.10.jar

This creates the initial files.

4. Accept EULA

Open eula.txt with notepad and change it to:

Open eula.txt with notepad
eula=true
Change to eula=true and save

Save and run minecraft_server.1.7.10.jar again. Close when it says "Done".

5. Run Forge

Double-click forge-1.7.10-10.13.4.1614-1.7.10-universal.jar

The forge universal jar file in your folder
Wait for "Done" in the console then close

6. Create Launch Script

Make a new file called start.bat and put this inside:

Right-click to create new batch file
Right-click the .bat file and select Edit
java -Xmx4096M -Xms4096M -XX:+UseG1GC -jar forge-1.7.10-10.13.4.1614-1.7.10-universal.jar PAUSE
This gives your server 4GB RAM. Change 4096M to use more/less.

7. Add Mods & Start

Put your mod .jar files in the mods folder.

Double-click start.bat to run your server.

Note: Friends can't connect yet - you need port forwarding or use the tool below.
